# Nixpkgs Configuration Knowledge Base

A local-first knowledge base for this multi-platform Nix configuration — macOS
(nix-darwin), NixOS, and WSL (standalone home-manager) built from a single flake. It
explains the patterns behind how the repo manages **modules** and **hosts**, for both
human readers and AI agents navigating with `iwe retrieve`.

Start with the architecture, then read modules and hosts. The guides are task-oriented;
the references are for lookup.
